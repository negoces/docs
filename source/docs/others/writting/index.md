## 提示信息

<https://doku.skk.moe/tag-plugins.html>

语法：

```js
{% [标签插件的名称] [可选的标签插件的参数] %}
[可选的标签插件的内容]
{% end[标签插件的名称] %}
```

样式列表：

- none
- dark
- primary
- link
- info
- success
- warning
- danger
